## Default Permission

Default permissions for the plugin `status-bar`

- `allow-background-color-by-hex-string`
- `allow-background-color-by-name`
- `allow-hide`
- `allow-is-visible`
- `allow-overlays-web-view`
- `allow-registerListener`
- `allow-show`
- `allow-style-default`
- `allow-style-light-content`
